# card-agent — โปรแกรมอ่านบัตรประชาชน (USB)

โปรแกรมเล็กๆ ที่รันในเครื่องพยาบาล เพื่ออ่านบัตรประชาชนจากเครื่องอ่าน USB (PC/SC)
แล้วส่งข้อมูลให้เว็บแอปกรอกฟอร์มอัตโนมัติ เมื่อกดปุ่ม **"อ่านบัตร"** ในหน้าลงทะเบียน

> **ทำไมต้องมีโปรแกรมนี้?** เบราว์เซอร์อ่านชิปบัตรโดยตรงไม่ได้ (เพื่อความปลอดภัย)
> โปรแกรมนี้ทำหน้าที่เป็น "ตัวกลาง" ระหว่างเครื่องอ่านบัตรกับเว็บ

---

## สิ่งที่ต้องมี
- เครื่องอ่านบัตรสมาร์ทการ์ด **มาตรฐาน PC/SC** (USB) — รุ่นทั่วไปในตลาดใช้ได้
- **Node.js** ติดตั้งในเครื่อง (เวอร์ชัน 16 ขึ้นไป) — https://nodejs.org
- Windows: มี "Smart Card" service (มากับ Windows อยู่แล้ว)

## ติดตั้ง (ทำครั้งเดียว)
```bash
cd card-agent
npm install
```

## เปิดใช้งาน (ทุกครั้งก่อนใช้)
```bash
npm start
```
ถ้าขึ้นข้อความ `card-agent พร้อมใช้งานที่ http://localhost:8765` = พร้อมแล้ว
(เปิดหน้าต่างนี้ค้างไว้ระหว่างใช้งาน)

## วิธีใช้
1. เสียบเครื่องอ่านบัตร USB + เสียบบัตรประชาชน
2. เปิดเว็บแอป → หน้า **ลงทะเบียนผู้รับบริการใหม่**
3. กดปุ่ม **"อ่านบัตร"** ข้างช่องเลขบัตรประชาชน
4. ระบบดึง เลขบัตร / ชื่อ-สกุล / วันเกิด / เพศ / ที่อยู่ / **รูปจากบัตร** มากรอกให้อัตโนมัติ
5. ตรวจสอบข้อมูล → กดบันทึก

---

## หมายเหตุทางเทคนิค
- โปรแกรมเปิด server ที่ `http://localhost:8765` เฉพาะในเครื่อง (127.0.0.1) — ปลอดภัย ไม่เปิดออกเน็ต
- เว็บที่รันบน HTTPS เรียก `http://localhost` ได้ (เบราว์เซอร์อนุญาตเป็นกรณีพิเศษ)
- ข้อความภาษาไทยในบัตรเข้ารหัส **TIS-620** โปรแกรมแปลงเป็น UTF-8 ให้แล้ว
- ถ้าเปลี่ยนพอร์ต ให้แก้ค่า `ID_CARD_AGENT_URL` ในไฟล์ `index.html` ด้วย

## แก้ปัญหาเบื้องต้น
| อาการ | วิธีแก้ |
|-------|---------|
| เว็บขึ้น "เชื่อมต่อเครื่องอ่านบัตรไม่ได้" | ยังไม่ได้ `npm start` หรือปิดหน้าต่างโปรแกรมไป |
| "ยังไม่ได้เสียบบัตร" | เสียบบัตรให้แน่น รอ 1-2 วินาที |
| `npm install` error (Windows) | ติดตั้ง "Windows Build Tools" / Visual Studio Build Tools ก่อน |
| อ่านช้า/ค้าง | บางเครื่องอ่านต้องถอด-เสียบบัตรใหม่ |

> โค้ดนี้เป็น **template อ้างอิง** ตามสเปกบัตรประชาชนไทยมาตรฐาน หากเครื่องอ่านรุ่นที่ใช้มีพฤติกรรมต่าง (เช่นต้องหน่วงเวลา) อาจปรับ `agent.js` เล็กน้อย — แจ้งรุ่นเครื่องอ่านมาได้ ปรับให้ตรงรุ่นได้
